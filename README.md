# Organization Community Health Files

This repository contains the centralized community health files for our organization's open source projects. These files provide standard guidelines and information for contributors across all repositories.

## Community Health Files

| File | Purpose |
| :--- | :--- |
| [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md) | Standard community guidelines and reporting procedures. |
| [CONTRIBUTING.md](./CONTRIBUTING.md) | Guidelines for contributing to our projects, including CLA requirements. |
| [SECURITY.md](./SECURITY.md) | Instructions for reporting security vulnerabilities. |

## Usage

These files are typically used as default community health files for all repositories in the organization when they don't provide their own versions in their respective `.github` or root directories.
